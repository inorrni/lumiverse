# 개발일지 — 2026.06.21

> [!note] 한 줄 요약
> 메인페이지(대시보드)를 중심으로 **표시·동선·핵심 루프**를 대거 다듬었다. ① 목표 카드에 마감일(`~YYYY.MM.DD · D-N`)+모드 태그, ② 대시보드 캐러셀을 **오늘 별 있는 목표만** 노출하고 **Daily Stars 분모 버그** 교정, ③ 우주탭에 **상태 필터**(진행중/기간종료/완료 + 중간점검 단계), ④ 별자리 안내 카피 + **생성 전 확인 단계(더 키우기/지금 만들기·개수 선택)**, ⑤ **메인 Today 카드에서 투두 체크·회고** 가능, ⑥ 로그인 후 기존 유저 **대시보드 직행**(K3 레이스 해소), ⑦ 설정 **닉네임 수정**, ⑧ 중간점검 아이콘 telescope 교체, ⑨ **행성상세 재클릭 무반응 버그** 수정. 더불어 README 전면 개편.

## 1. 배경

0620까지 핵심 루프·블랙홀이 정리되며 대시보드가 메인 허브가 됐다. 사용 점검에서 메인페이지에 다음이 보고됐다.
- 목표 카드에 **마감 날짜가 안 보임**(제목만).
- 별자리 안내가 짧은 목표엔 도달 불가능한 약속("N개 더 모으면")을 함.
- **디데이 지난 목표 처리 부재** → 무기한과 동일하게 '∞' 표기, 대시보드 캐러셀도 목표가 많아지면 감당 불가.
- 최상단 **Daily Stars 총합에 기간종료 목표의 행성까지 잡혀** 100% 도달 불가.
- 로그인 후 정보 있는 유저도 온보딩(`/mode`)으로 빠짐(K3).
- 설정에서 **닉네임 수정 불가**, 메인 Today는 **읽기 전용**.
- 행성상세 목록에서 행성 클릭 → 진입 → 뒤로 → **재클릭 무반응**.

## 3. 핵심 설계 메모

> [!important] 파생으로 끝낸 라이프사이클
> 기간종료/완료는 **DB 상태 추가 없이** `deriveGoal`에서 파생(`expired = dday_end < today`, `completed = clarity===100`). 블랙홀만 실제 `status`. 덕분에 우주탭 필터·대시보드 노출 모두 마이그레이션 0으로 구현.

> [!important] 대시보드 ↔ Daily Stars 일관
> 대시보드를 "오늘 별 있는 목표만"으로 좁히고 Daily Stars 분모도 같은 기준(오늘 별 있는 행성)으로 맞춰, **표시 대상과 카운트가 한 기준**에서 나오도록 통일(0620의 Today 카드 단일 소스화와 같은 결).
